# TPE virtuel

## Utilisation
Double cliquer sur le fichier index.html.

## Description

Ceci est TPE virtuel réalisé en HTML, CSS et JavaScript. Il permet de simuler un paiement de 20 euros, il est possible de :
- saisie un code. Si le code saisie est 1234, l'utilisateur peut simuler si la transaction est réussie ou échouée via une pop up.
- de revenir en arrière.
- d'annuler la transation.
